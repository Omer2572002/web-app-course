⚙️ الوحدة 3: تصنيفات تطبيقات الويب

  1. التطبيقات الثابتة (Static Web Apps)

    يتم عرض نفس المحتوى لجميع المستخدمين، بغض النظر عن هويتهم أو الوقت. يتم تخزين الملفات (HTML, CSS, JS) بشكل كامل على الخادم ولا تتغير إلا بالتحديث اليدوي للمطور.

  2. التطبيقات الديناميكية (Dynamic Web Apps)

    هي النوع الأكثر شيوعاً، حيث يتغير المحتوى بناءً على تفاعل المستخدم أو البيانات المخزنة في قاعدة البيانات. تتطلب معالجة على جانب الخادم (Backend) لإنشاء صفحة HTML المطلوبة في الوقت الحقيقي.

  3. تطبيقات الصفحة الواحدة (Single Page Applications - SPA)

    يتم تحميل كود الـ HTML الأساسي مرة واحدة فقط، ثم يتم تحديث المحتوى بشكل ديناميكي باستخدام **JavaScript** دون الحاجة لإعادة تحميل الصفحة بالكامل. هذا يوفر تجربة سريعة وسلسة (مثل تطبيقات Gmail أو خرائط Google).
